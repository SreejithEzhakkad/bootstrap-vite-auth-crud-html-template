Auth & CRUD HTML Template

This is a simple HTML template created using Bootstrap 5.3, and Vite. This template contains authentication and CRUD pages.

Get Source Code

Installation

Clone the repository.
$ git clone git@github.com:SreejithEzhakkad/bootstrap-vite-auth-crud-html-template.git

Change the values in .env file(Optional) and do the following
$ npm install
$ npm run dev

To create build for production
$ npm run build

To preview the production build.
$ npm run preview
